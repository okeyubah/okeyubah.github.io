The purpose of this website is to teach Nigerian children Java, starting from SS1 (Senior Secondary 1). To this end, we have tried very hard to keep our English simple.

To introduce young Nigerians to Java, we will be using "A Peek into Java for Nigerian Teenagers", a book that the Lagos State Ministry of Education recently approved for use in senior secondary schools. The author of that book, Anthony Okechukwu Ubah, also happens to be the owner of this blog.

In other words, this website was set up to help the author of "A Peek into Java for Nigerian Teenagers" explain the contents of his book, for the benefit of interested teachers and pupils.
