where teenagers start ...to learn Java

Pupils in SS1 (Senior Secondary 1) can now start reading "A Peek into Java for Nigerian Teenagers"

This book was written by the owner of this blog, and

  • was recently approved for SS1 pupils by the Lagos State Ministry of Education
  • will be explained in great detail by its author, Anthony Okechukwu Ubah
  • could eventually lead the pupil to a lucrative role as a computer programmer

The objective:

  • explain every new word that helps to mystify the language of computer programmers
  • take pupils slowly through the fundamental aspects of the Java programming language
  • convince parents that Java should be an important part of their children's curriculum

To get started, click the Staff link below, then select the posts that were written by this author. You can also click the Topics link below, to select any topic from the book.
